Retour Identity Management (IAM)

Gestion des identités : Microsoft Entra ID

Automatisation de l'onboarding utilisateur et des groupes de sécurité via Terraform.

ETAPE 1

Connexion à l'Active Directory Azure

Configuration du provider azuread pour intéragir avec l'annuaire de TiersLieux86.

terraform {
  required_providers {
    azuread = { source = "hashicorp/azuread" }
  }
}
[ICI : Capture de ton code VS Code ou de l'authentification az login]
ETAPE 2

Provisionnement des comptes

Création standardisée des comptes collaborateurs avec génération de mot de passe temporaire.

resource "azuread_user" "user" {
  user_principal_name = "j.dupont@tierslieux86.fr"
  display_name = "Jean Dupont"
  password = "InitialPassword123!"
}
[ICI : Capture de la liste des utilisateurs dans Entra ID]
ETAPE 3

Groupes de Sécurité & Rôles

Assignation automatique des utilisateurs à des groupes pour la gestion des droits (RBAC).

resource "azuread_group" "it_staff" {
  display_name = "Equipe IT"
  security_enabled = true
}
