07RSTP · Rapid-PVST+
Redondance L2 — RSTP
Spanning Tree rapide et optimisation convergence
Complété

Remplacement du STP classique par Rapid-PVST+ pour réduire le temps de convergence de 30-50s à moins de 5s. SwitchIntra est élu Root Bridge pour tous les VLANs.

Passage en mode rapid-pvst sur tous les switches.
SwitchIntra élu Root Bridge (priorité 4096) pour tous les VLANs.
PortFast activé sur les ports d'accès pour éliminer le délai de démarrage.
BPDU Guard activé globalement pour sécuriser les ports PortFast.
SwitchIntra — Rapid-PVST+ Root Bridge
IOS
! Activation Rapid-PVST+ sur tous les switches
SwitchIntra(config)# spanning-tree mode rapid-pvst

! SwitchIntra = Root Bridge pour tous les VLANs
SwitchIntra(config)# spanning-tree vlan 1-99 root primary

! PortFast + BPDU Guard sur ports d'accès
SwitchEtages(config)# spanning-tree portfast default
SwitchEtages(config)# spanning-tree portfast bpduguard default

! Vérification
SwitchIntra# show spanning-tree vlan 10

CAPTURE — show spanning-tree vlan 10

08HSRP · L3
Redondance L3 — HSRP
Haute disponibilité de la passerelle par défaut
Complété

Mise en place de HSRP entre RoutChass (actif, priorité 110) et RouterFAI (standby). En cas de panne du routeur actif, le basculement est automatique et transparent pour les utilisateurs.

RoutChass = routeur actif (priorité 110) avec preempt pour reprendre le rôle à la restauration.
RouterFAI = routeur standby (priorité 100 par défaut).
IP virtuelle partagée : 192.168.2.254 — configurée comme passerelle sur tous les postes clients.
Test : extinction de RoutChass → RouterFAI prend le relais sans interruption.
RoutChass — HSRP Actif / RouterFAI — Standby
IOS
! RoutChass = routeur HSRP actif (priorité 110)
RoutChass(config)# interface Fa0/0
RoutChass(config-if)# ip address 192.168.2.1 255.255.255.0
RoutChass(config-if)# standby 1 ip 192.168.2.254
RoutChass(config-if)# standby 1 priority 110
RoutChass(config-if)# standby 1 preempt

! RouterFAI = routeur HSRP standby (priorité 100)
RouterFAI(config)# interface Fa0/0
RouterFAI(config-if)# ip address 192.168.2.2 255.255.255.0
RouterFAI(config-if)# standby 1 ip 192.168.2.254
