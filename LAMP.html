Infrastructure Web : LAMP & WordPress

Installation d'un serveur Debian sécurisé pour l'hébergement du site CMS de TiersLieux86.

ÉTAPE 1

Services Apache2 & PHP

Installation du serveur HTTP et du moteur de script pour le rendu dynamique.

sudo apt update && sudo apt install apache2 php libapache2-mod-php php-mysql -y
# Vérification du service
sudo systemctl status apache2
ÉTAPE 2

Base de données MariaDB

Création de la base de données et de l'utilisateur dédié à WordPress.

sudo mysql_secure_installation
# Création DB
sudo mysql -u root -p
CREATE DATABASE wordpress_db;
CREATE USER 'wp_user'@'localhost' IDENTIFIED BY 'MonMotDePasseFort';
GRANT ALL PRIVILEGES ON wordpress_db.* TO 'wp_user'@'localhost';
ÉTAPE 3

Déploiement de WordPress

Téléchargement et configuration des droits sur le répertoire /var/www/html/.

wget https://wordpress.org/latest.tar.gz
tar -xvf latest.tar.gz
sudo cp -R wordpress/* /var/www/html/
sudo chown -R www-data:www-data /var/www/html/
